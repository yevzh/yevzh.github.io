代理服务器

第一天做了一条难题，代理服务器，n个代理，m个服务器，按照m个服务器的顺序来选择代理，使得切换代理最少。我们的第一想法就是贪心呐，选择一个代理使得第一次能有尽量多的服务器挂在上面。

题解中有一个比喻比较好，漏筛。我们让服务器依次过代理，哪个重合就将其设为false，直到全部都堵上，那么最后一个堵上的就是我们要选择的第一个代理，这时候切换次数++。

关键代码如下：

for(int i=0;i<m;i++)
   {
       for(int j = 0;j<n;j++)
       {
           if(agents[j]==servers[i])
           {
               if(agents.size()==1)
               {
                   cout<<-1<<endl;
                   return 0;
               }
               flag[j] = true;
               bool ifall = true;
               // cout<<"hello"<<endl;
               // cout<<j<<endl;
               for(int k=0;k<n;k++)
               {
                   if(flag[k]==false)
                   {
                       ifall = false;
                       break;
                   }
               }
               if(!ifall)
               {
                   break;
               }
               // if all full
               for(int k=0;k<n;k++)
               {
                   if(k!=j)
                   {
                       flag[k]=false;
                   }
               }
               cnt++;
               break;
           }
       }
   }

全排列

思想就是分治递归。具体字符串怎么递归呢？首先选一个循环，让每个字符都有可能在第一位，然后分别在每个字符在第一位的情况下进行递归。也就是说给第二位选择排列，这个意思。

void ranking(string prefix, string substring)
{
    if(substring.size()==1)
    {
        // cout<<prefix<<substring<<endl;
        dict.push_back(prefix+substring);
        return;
    }
    for(int i=0;i<substring.size();i++)
    {
        string prefix_new = prefix+substring[i];
        string tmp = substring;
        string substring_new = tmp.erase(i,1);
        ranking(prefix_new, substring_new);
    }
}

BFS

BFS的思想就是立即扩展周围的所有情况，并放入维护的队列中。如何把一个状态的下一个状态push进队列是问题的关键。

做了一条题目是玛雅人的密码，经过几次move能让它变成符合要求的串？这样我们知道，需要用一个结构体，存放move的次数和当前的串。每一次我们swap一下相邻的字符然后把它push进队列，就作为下一个状态。在这里有两个操作需要背一下：

  • 寻找字串是否在给定串里面：

    if(t.s.find("2012")!=string::npos)

  • 交换字符串的某两个个字符，直接用swap

DFS

这类问题感觉很重要，这边就根据自己做的一题整理一下。神奇的口袋，问怎么选可以选中刚好40体积的物品。这个问题有点像01背包，就是我应该选哪些。这就是一个搜索问题了。具体的搜索函数怎么写？

首先定义终止条件，如果等于40就计数+1退出函数。如果大于40直接退出。如果小于四十，我们直接递归下去，这个时候我们要找状态转移，我第一个要选哪一个呢？

我可以用一个循环，第一个我可以选，也可以不选；我可以前两个都不选……

void dfs(int sum, int index, int n)
{
    if(sum==40)
    {
        cnt++;
        return;
    }
    else if(sum>40) return;
    else
    {
        for(int i=index; i<n;i++)
        {
            dfs(sum+a[i], i+1, n);
        }
    }
}

这边就顺便提一下一条更难的题目，给一个数组如何分成最接近一半一半的。

首先判断下标，如果超了就return。

然后如果小于等于一半的时候，我们可以选择添加这个，也可以选择不添加这个，所以我们加入这个，进行下一步递归，同时在判断外面我们总得有不加入这个的递归：

void dfs(int sum, int index) {
    if (index >= number)return;
    if ((sum + arr[index]) <= totalsum / 2) {
        res = max(sum + arr[index], res);
        if (res == totalsum / 2)return;
        dfs(sum + arr[index], index + 1);
    }
    dfs(sum, index+1);
}

基于这个启发，我们可以改进刚才的神奇的口袋：

void dfs(int sum, int index, int n)
{
    if(index>n)return;
    if(sum==40)
    {
        cnt++;
        return;
    }
    else if(sum>40) return;
    else
    {
        dfs(sum+a[index], index+1,n);
    }
    dfs(sum, index+1, n);
}

二叉树

首先二叉树的建树是比较八股的东西，首先定义TreeNode：

struct TreeNode{
    char data;
    struct TreeNode *lchild, *rchild;
    TreeNode(char c): data(c), lchild(NULL), rchild(NULL){};
};

比如我们根据一个字符串建立一个树，根据字符串的顺序：

TreeNode * createTree(string str)
{
    char c = str[create_pos++];
    // cout<<create_pos;
    if(c=='#')
    {
        return NULL;
    }
    TreeNode *root = new TreeNode(c);
    root->lchild= createTree(str);
    root->rchild = createTree(str);
    return root;
}

还有二叉排序树的插入操作：

TreeNode * insert(TreeNode*root, int x)
{
    if(!root)
    {
        root = new TreeNode(x);
    }
    else if(x<root->data)
    {
        root->lchild=insert(root->lchild,x);
    }
    else if(x>root->data)
    {
        root->rchild=insert(root->rchild, x);
    }
    return root;
}

也是一个递归建树的过程。

并查集

并查集用来查看不同的节点最后归于了几类，一个比较常见的就是看一个图的连通分量。

并查集更多像个模板题，模板需要背：

int Find(int x)
{
    if(x!=father[x])
    {
        father[x] = Find(father[x]);
    }
    return father[x];
}
 
void Union(int x, int y)
{
    x = Find(x);
    y = Find(y);
    if(x!=y)
    {
        if(height[x]<height[y])
        {
            father[x] = y;
        }
        else if(height[x]>height[y])
        {
            father[y]=x;
        }
        else {
            father[y] = x;
            height[x]++;
        }
    }
}

Father 和 Height，Find和Union。
